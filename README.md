Music Player App

Android | Kotlin | XML | MVVM

Este projeto é um Music Player para Android desenvolvido utilizando Kotlin e XML, seguindo o padrão de arquitetura MVVM (Model–View–ViewModel) para manter o código organizado, escalável e de fácil manutenção.

O objetivo deste projeto é demonstrar boas práticas no desenvolvimento Android, incluindo separação de responsabilidades, organização de recursos e criação de interfaces modernas.

Stack Utilizadas

Kotlin

XML (Layouts Android)

Android Studio

MVVM Architecture

Android Jetpack Components

MediaPlayer API

 Features

🎵 Reprodução de músicas

⏯️ Controles de play, pause e próxima faixa

📂 Organização de recursos e layouts

🎨 Interface moderna construída com XML

🧩 Arquitetura MVVM para melhor estrutura do código


🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

Praticar desenvolvimento Android moderno

Aplicar o padrão MVVM

Criar uma UI funcional e organizada

Demonstrar habilidades em Kotlin e arquitetura de apps
